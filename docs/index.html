CallerInfo Protocol

Lightweight, efficient data protocol that compresses caller information into small BLE advertisement payloads. Includes phone number, caller name, and timestamp data.

BLE Serialization

Binary serialization optimized for the 31-byte BLE advertisement data limit, using compression techniques to maximize information transfer.

Getting Started

Set up your environment and start using CallerBLE

Prerequisites

  • Java 25 installed and JAVA_HOME set
  • Bluetooth hardware enabled on both devices
  • Android SDK and/or Xcode for mobile builds
  • Windows 10, macOS, or Linux desktop
  • Git to clone repository

Quick Start Commands

Clone Repository

git clone https://github.com/your-org/callerble.git

Build Project

./gradlew clean build --warning-mode all

Run Desktop Client

./gradlew :desktop:run

Build for Android

./gradlew :mobile:gluonfx:package -Ptarget=android

Build for iOS

./gradlew :mobile:gluonfx:package -Ptarget=ios
